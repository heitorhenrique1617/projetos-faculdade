# Receitas da Clara 

Este é um projeto pessoal de um portal de culinária simplificada, mantido neste repositório como parte do meu portfólio de aprendizado em desenvolvimento Web.

## 📝 Sobre o Projeto
O "Receitas da Clara" foi idealizado para oferecer instruções claras, concisas e acessíveis para quem deseja cozinhar sem complicações. O design utiliza tons pastéis e tipografia amigável para refletir a simplicidade das receitas propostas.

## 🛠️ Tecnologias e Recursos
* **HTML5**: Estrutura das páginas de Home, Sobre e Categorias.
* **CSS3**: Estilização personalizada, incluindo:
    * **Google Fonts**: Uso da fonte "Poetsen One".
    * **Responsividade**: Media queries para adaptação em tablets e celulares.
    * **Efeitos Visuais**: Filtros de brilho e escala no hover das imagens.
* **JavaScript**: Integração de scripts para funcionalidades futuras.

## 🚧 Status do Projeto
**Incompleto / Em desenvolvimento.** O projeto serve atualmente como estudo de navegação entre múltiplas páginas HTML e aplicação de Flexbox em menus e galerias.

## 📂 Estrutura de Arquivos
* `index.html`: Página principal com categorias.
* `sobre.html`: História e objetivos do site.
* `stylemenu.css`: Folha de estilo principal com foco em responsividade.

*Projeto desenvolvido para fins de estudo de desenvolvimento Web.*
