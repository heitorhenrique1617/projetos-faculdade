# Site Institucional - Heitor Serviços

Este é um projeto de um site institucional moderno e totalmente responsivo, focado em prestação de serviços de tecnologia e marketing digital.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização com Layout de Grid, Flexbox e animações de transição.
* **JavaScript**: Lógica para menu hamburguer (mobile) e submenus interativos.

## 📱 Funcionalidades
* **Menu Responsivo**: Adaptável para dispositivos móveis e desktops.
* **Dropdown Interativo**: Submenus para organização de categorias (Serviços e Produtos).
* **Galeria de Projetos**: Grid de imagens com efeito de zoom (*hover*).
* **Rodapé Dinâmico**: Posicionamento fixo na base da página.
